# ⚡ Inicio Rápido

## 🚀 En 5 Minutos

### Para XAMPP (Local)

1. **Copiar archivos**
   ```
   C:\xampp\htdocs\CURRICULUM\
   ```

2. **Iniciar Apache**
   - Abre XAMPP Control Panel
   - Haz clic en "Start" en Apache

3. **Acceder**
   ```
   http://localhost/CURRICULUM/
   ```

### Para GitHub Pages (En Línea)

1. **Crear repositorio**
   - Ve a github.com
   - Crea nuevo repositorio: `CURRICULUM`

2. **Subir archivos**
   ```bash
   git clone https://github.com/TU_USUARIO/CURRICULUM.git
   cd CURRICULUM
   # Copia los archivos aquí
   git add .
   git commit -m "Currículum inicial"
   git push origin main
   ```

3. **Habilitar GitHub Pages**
   - Settings → Pages
   - Source: main branch
   - Accede a: `https://TU_USUARIO.github.io/CURRICULUM/`

## 🆘 Problemas Comunes

### Apache no inicia
- Ejecuta XAMPP como administrador
- Verifica que el puerto 80 esté libre

### Página no carga
- Verifica la ruta: `C:\xampp\htdocs\CURRICULUM\`
- Limpia caché del navegador (Ctrl+Shift+Delete)

### Foto no se ve
- Asegúrate que el archivo esté en la carpeta
- Verifica el nombre exacto del archivo

### GitHub Pages no funciona
- Espera 1-2 minutos después de subir
- Verifica que GitHub Pages esté habilitado en Settings
