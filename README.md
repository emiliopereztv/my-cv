## 🚀 Instalación Local (XAMPP)

### Requisitos
- XAMPP instalado en tu sistema
- Navegador web moderno

### Pasos

1. **Clonar o descargar el repositorio**
   ```bash
   git clone https://github.com/EmilioPerezTV/CURRICULUM.git
   ```

2. **Copiar archivos a XAMPP**
   ```
   C:\xampp\htdocs\CURRICULUM\
   ```

3. **Iniciar XAMPP**
   - Abre el Panel de Control de XAMPP
   - Inicia Apache

4. **Acceder a la página**
   ```
   http://localhost/CURRICULUM/
   ```

## 🌐 Despliegue en GitHub Pages

### Pasos

1. **Crear un repositorio en GitHub**
   - Nombre: `CURRICULUM` o `EmilioPerezTV.github.io`

2. **Configurar GitHub Pages**
   - Ve a Settings → Pages
   - Selecciona la rama `main` como fuente
   - Guarda los cambios

3. **Subir archivos**
   ```bash
   git add .
   git commit -m "Currículum profesional inicial"
   git push origin main
   ```

4. **Acceder a la página**
   ```
   https://EmilioPerezTV.github.io/CURRICULUM/
   o
   https://EmilioPerezTV.github.io/
   ```
